# 路线图

## 版本计划

### v0.7 — Agent 系统 ✅（第一阶段已完成）
- 完成 Agent 相关功能
- MCP、Computer Use、Browser Use、虚拟机集成

### v0.8 — 记忆、桌宠与小游戏 ✅（第一阶段已完成）
- 增强记忆管理
- 桌宠模式
- 内置多款双人小游戏
- 改进记忆压缩与召回

### v0.9 — 多平台（目标：2026 年 7 月初）
- Linux 支持改进
- 移动平台探索
- N.E.K.O. Network 启动（AI 社交功能）

### v1.0 — 自定义 AI（目标：2026 年 8 月底）
- 放弃部分第三方模型服务商的适配
- 转向自研大模型
- 专用 Agent 系统

## 长期愿景

### 第一阶段：创意工坊（当前）
Steam 上的核心引擎，配合创意工坊支持共享自定义内容。

### 第二阶段：独立平台
独立应用和网站，拥有更丰富的 UGC 社区以及 AI 原生游戏生态系统。

### 第三阶段：N.E.K.O. Network
自主 AI 社交化——角色拥有自己的"意识"，能够交流、组建群组，并在模拟社交媒体上发帖。
