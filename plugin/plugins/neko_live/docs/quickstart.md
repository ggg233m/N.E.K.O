# NEKO Live 快速开始

这份教程给第一次使用的主播看。照着做就行，不需要懂开发环境。

## 开播前

1. 打开打包版 N.E.K.O。
2. 进入“插件管理”。
3. 打开 “NEKO Live” 插件详情。
4. 如果插件详情显示“启动”，先启动插件；当前 manifest 使用 `auto_start=false`，未启动时面板接口不可用。
5. 切到“面板”。

## 连接直播间

1. 在“控制台”里选择直播平台。
2. 点击“管理账号与平台”并登录：B 站推荐扫码登录，抖音导入凭据。B 站无账号连接只作为当前会话手动启用的兼容性兜底。
3. 打开“确认直播间”，填直播间号或直接粘贴直播间链接。
4. 先点“查询直播间”，核对房间标题、主播名和直播状态；确认无误后再点“确认直播间”。修改输入后必须重新查询，旧结果不会继续生效。
5. 控制台底部的“开始直播”会在账号和直播间都准备好后变为可用。点击它开始监听。
6. 顶部状态显示“直播中”或事件显示“接收中”，就说明直播间事件通道已经接上。

## 让 NEKO 开口

1. 先让直播间发一条测试弹幕，比如“猫猫在吗”。
2. 看 NEKO 是否回复。
3. 如果回复了，就可以正常直播。
4. 如果没回复，点“刷新状态”，再看面板顶部状态：
   - “不能开播”：先检查直播间号、是否开始监听、输出通道是否可用。
   - “已阻塞”或“未连接”：重新开始监听。
   - “冷却中”或“等待”：等几秒，再发一条明确的问题测试。

## 常用位置

- “控制台”：管理账号、查询并确认直播间、开始 / 结束直播、切换直播模式和节奏设置。直播主题与节奏弹窗中的修改只会在点击“保存设置”后生效；直接关闭或取消会丢弃本次草稿。
- “直播间互动”：管理首次出场锐评、后续弹幕接话、礼物致谢和主动主持；每个模块的详细参数在弹窗里调整，关闭模块不会让卡片布局跳动。
- “观众”：分为“本场直播”和“观众档案”，查看本场统计、最近互动观众和安全画像。点开单个观众可“重置印象”或“删除档案”：重置只忘记偏好，删除则让该 UID 下次按新观众处理。技术链路与最近 pipeline 结果不放在普通观众页。
- “设置”：分为“安全与性能 / 数据与隐私 / 帮助与高级”。安全暂停建议保持开启；忙时处理用“谨慎 / 标准 / 宽松”三档，默认推荐“标准”。“个性化观众记忆”默认开启，只学习安全派生偏好、不保存原始弹幕，90 天未互动会自动清理；关闭后仍保留本场防复读和首次出场判断。这里也可以清空全部观众档案、查看本地档案位置、重新打开新手教程或启用开发者模式。
- “开发者工具”：只有开启开发者模式后出现，提供身份查询、模拟事件和运行结果。普通开播不需要进入这里。

## 直播主题怎么填

直播主题写一句普通、具体、不会泄露活动信息的话。它会帮助 NEKO 记住本场直播在聊什么。

可以这样写：

- 今天轻松聊天，和观众聊游戏、日常和最近在玩的内容。
- 本场是杂谈直播，重点是接弹幕、陪观众聊天，避免聊展会和未公开安排。
- 今天主打轻松陪伴，观众问什么就自然接，不主动提保密内容。

不要写：

- 展览、项目组、未公开活动、内部安排。
- 太长的剧本。
- 需要 NEKO 扮演真人主播、工作人员或后台操作员的内容。

## 直播中如果出问题

- 弹幕很多但 NEKO 不回：先等几秒，NEKO 会优先挑更值得回的弹幕。
- NEKO 回复太频繁：到“控制台”的“节奏设置”里降低说话频率或加长冷却时间。
- NEKO 太安静：发一条明确问题，比如“猫猫今天想聊什么”。
- “开始直播”点不了：先看顶部的“不能开播 / 未连接”提示，依次完成账号和直播间确认；按钮不会绕过缺失前置条件。
- 房间接不上：结束直播，再重新查询并确认直播间，然后重新开始。
- 不确定状态：先点“刷新状态”并查看“控制台”；需要链路解释时，在“设置 → 帮助与高级”开启开发者模式，再到开发者子界面查看运行结果。

## 下播后

1. 点击“结束直播”并确认。
2. 确认监听状态不再是“接收中”。
3. 需要继续测试时，再重新查询直播间并开始监听。
